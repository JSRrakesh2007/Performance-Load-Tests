## Project Scope

This project is developed using Python to perform a basic performance load test on a website. The program sends multiple HTTP GET requests to the target URL and records the response time for each request. At the end of execution, it calculates the average response time and saves the results to a text file.

## Technologies Used

- Python
- Requests Library

## Features

- Sends multiple requests to a website
- Measures response time
- Displays HTTP status code
- Calculates average response time
- Saves test results to a file

## How to Run

1. Install Python.
2. Install the required library.

```bash
pip install requests
```

3. Run the program.

```bash
python load_test.py
```

## Output

The program displays:

- Status Code
- Response Time
- Average Response Time

The summary is also saved in `results.txt`.
